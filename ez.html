Ellenállás

Fogalma

Az anyag azon tulajdonsága, hogy akadályozza a szabad töltéshordozók áramlását.

Kiszámítása: R = U/I, R = 1/G

Vezetőképesség

Az ellenállás reciproka.

Kiszámítása: G = I/U, G = 1/R

Eredő ellenállás

Bármelyin bonyolult ellenállás hálózat helyettesíthető egy darab ellenállással.

Sorosan kapcsolt ellenállások(a)

Re = R1 + R2 + R3 + R4

Párhuzamosan kapcsolt ellenállások(b)

Replusz művelet: Re = R1 x R2 = (R1 * R2) / (R1 + R2)

Re = R1 x R2 x R3 x R4

VAGY

Ge = G1 + G2 + G3 + G4

Ohm törvény

Egy zárt rendszerben az áramköri elemeken eső feszültség és folyó áram hányadosa egy állandó érték, mely jellemző a fogyasztóra.

R = U/I = állandó

Kirchhoff I.-II. törvénye

I. törvénye (csomóponti törvény): Egy csomópontba befolyó áramok összege megegyezik a csomópontból
kifolyó áramok összegével. Először fel kell venni egy körüljárási irányt, majd

Ablakhurok: Olyan hurok, ami nem tartalmaz több hurkot.

0 = I1 + I2 - I3 - I4

II. törvénye (hurok törvény): Egy zárt hurokban az áramköri elemeken eső feszültségek előjeles összege nulla.

0 = UR1 + UR2 + UR3 + UR4 - UR5 - U1 + U2 - U3 - U4


Norton-Thevenin ekvivalencia

Norton generátor (jobb oldali)

0 = I0 - Ib

Ib = Uk * Gb

Ik = -Uk * Gb + I0

Thevenin generátor (bal oldali)

0 = U0 - Ub

0 = Ik * Rb + Uk - U0

Ik = - 1/Rb * Uk + U0/Rb

Ekvivalencia

I0 * Rb = U0 = I0/Gb

Maximum teljesítmény

Rtmax = Rb

Ptmax = (U02 * Rb) / (Rb + Rt)2 = U02 / (4 * Rb2)

η = Rt / (2 * Rt) = 50%

Szuperpozíció tétele

Ha egy kapcsolásban több generátor is található, a hatásaikat külön vizsgáljuk meg, majd összegezzük őket.
